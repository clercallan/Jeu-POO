==== JEU ROGUE ====
Ce jeu a été développé dans le cadre d'un projet d'études en 2021. Il y a des bugs et le jeu n'est pas parfait. 

(!) Avant de lancer le jeu : 
=> Il est préférable de lancer le jeu sur Visual Studio Code en 1920 x 1080 et, par souci de place sur Moodle, de télécharger le fichier zip suivant et extraire le fichier musique jeu dans le même dossier que LeRogue.py (il s’agit des versions longues des musiques), sinon il y a des chances que la musique s’arrête dans le jeu. Si vous le faites, écrasez le fichier ‘musique jeu’ du rendu de base et remplacez-le par le fichier ‘musique jeu’ du zip. 
https://drive.google.com/file/d/12CtH2s3zUJpf9lfc5p0r5CANuPTF952G/view?usp=sharing

=> Il faut installer les modules PIL et playsound. Il est peut-être nécessaire de créer un environnement virtuel pour installer ces modules.

Instructions : 
Ce jeu est un jeu de plateformes, vous devez déplacer le joueur sur de petites "maps", et monter d'étage en étage. Il n'y a pas de fin, le but est d'aller le plus loin possible. Sur votre chemin vous rencontrerez des ennemies, de la nourriture, des armes qui augmentent votre puissance d'attaque et des potions qui ont différents pouvoirs. Bonne chance ! 

Touches : 
Z : Avancer
Q : Aller à gauche
S : Reculer
D : Aller à droite
Flèche du haut / bas : Naviguer dans l'inventaire
C : Consommer l'item sélectionné dans l'inventaire.

==== JEU ROGUE ==== (en)
(!) Before launching the game:
=> It is preferable to run the game in Visual Studio Code at 1920 x 1080 resolution, and, due to space constraints on Moodle, download the following zip file and extract the game music file into the same folder as LeRogue.py (these are the extended versions of the music). Otherwise, there's a chance the in-game music might stop. If you do this, replace the 'game music' file from the original submission with the 'game music' file from the zip.
https://drive.google.com/file/d/12CtH2s3zUJpf9lfc5p0r5CANuPTF952G/view?usp=sharing

=> You need to install the PIL and playsound modules. It may be necessary to create a virtual environment to install these modules.

Instructions:
This game is a platformer where you must move the player through small "maps" and ascend from floor to floor. There's no end; the goal is to go as far as possible. Along the way, you will encounter enemies, food, weapons that increase your attack power, and potions with various powers. Good luck!

Controls:
Z: Move forward
Q: Move left
S: Move backward
D: Move right
Up/Down arrow: Navigate the inventory
C: Consume the selected item in the inventory.
